Pure CSS accordion component with animation

What features are used?
  • CSS calc-size()
  • CSS transition-behavior: allow-discrete
  • CSS ::details-content
  • Exclusive Accordion with <details name="">
How it Works?
calc-size() can automatically animate the height from 0 to auto.
Browser compatibility
Currently only supported by Chrome, you need to enable flag in Canary.

chrome://flags/#enable-experimental-web-platform-features

Another way to hack is to use the Grid's fr unit.
